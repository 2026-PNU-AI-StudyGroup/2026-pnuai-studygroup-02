## 2. `data/ingredient_aliases.json` (동의어 매핑 사전)
* **목적:** AI 모델 출력 영문명 및 사용자 입력 유의어를 `nutrition.csv`의 기준 한글명(`ingredient_name`)과 1:1로 매핑하기 위한 파일
* **구조:** `{"입력될 단어": "CSV 기준 단어"}` 형태의 평면(Flat) 구조 
* **진행 상황:** 위 20개 식재료 각각에 대해 '영문 클래스명 1개 + 한글 동의어/오타 5개'씩 매핑 추가 완료
